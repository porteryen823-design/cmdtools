### 使用指南

#### 1. 環境準備
```bash
# 安裝依賴套件
pip install -r requirements.txt

# 或使用國內鏡像源
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/
```

#### 2. 設定資料庫
編輯 `config.json` 檔案，設定您的 MySQL 連線資訊：
```json
{
  "DBServer": "127.0.0.1",
  "DBPort": "3306",
  "DBUser": "您的用戶名",
  "DBPassword": "您的密碼",
  "DataBase": "MyCmdTools"
}
```

#### 3. 啟動程式
```bash
python main.py
```

#### 4. 基本操作
- **切換分頁**: 點擊 "命令工具" 或 "提示工具" Tab
- **搜尋資料**: 使用全域搜尋或單一欄位搜尋
- **編輯資料**: 點擊 "新增"、"編輯"、"刪除" 按鈕
- **匯出資料**: 點擊 "匯出" 按鈕選擇格式和範圍
